Ejercicio Sueldos

Que calcule el sueldo que le corresponde al trabajador de una empresa que cobra 20.000 pesos mensuales, el programa debe realizar los cálculos en función de los siguientes criterios: a. Si lleva más de 10 años en la empresa se le aplica un aumento del 10%. b. Si lleva menos de 10 años pero más que 5 se le aplica un aumento del 7%. c. Si lleva menos de 5 años pero más que 3 se le aplica un aumento del 5%. d. Si lleva más de 6 meses pero menos de 3 años se le aplica un aumento del 3%.

#include <stdio.h>
#include <conio.h>
#define sueldo (20000)
void main()
{
    int anios,aumento,sueldofin;
 
        printf("Ingrese cantidad de anios trabajados : ");
        scanf("%d", &anios);
 
        if (anios>=10)
        {
            aumento = sueldo * 10 / 100;
            sueldofin = sueldo + aumento;
            printf("El aumento es de: %d y el sueldo final es de : %d",aumento,sueldofin);
        }
        else if ( anios < 10 && anios >= 5)
        {
            aumento = sueldo * 7 / 100;
            sueldofin = sueldo + aumento;
             printf("El aumento es de: %d y el sueldo final es de : %d",aumento,sueldofin);
        }
        else if ( anios < 5 && anios >= 3 )
        {
            aumento = sueldo * 5 / 100;
            sueldofin = sueldo + aumento;
             printf("El aumento es de: %d y el sueldo final es de : %d",aumento,sueldofin);
        }
        else if ( anios < 3 )
        {
            aumento = sueldo * 3 / 100;
            sueldofin = sueldo + aumento;
             printf("El aumento es de: %d y el sueldo final es de : %d",aumento,sueldofin);
        }
 
        getch();
 
}
